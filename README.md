# Keytap Analyzer
Tool for Measuring Keyboard Typing Sound

## Requirements
### Hardware
- Mic
    - behringer ECM PRO
- Audio Interface
    - Shteinberg UR22C

### Software
- **Browser** (with Web Audio API & getUserMedia support):
  | Browser | Minimum Version |
  |---------|-----------------|
  | Chrome | 53+ |
  | Firefox | 36+ |
  | Safari | 14.1+ |
  | Edge | 79+ |
  | Opera | 40+ |
